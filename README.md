📊 Netflix Data Analysis Dashboard (Excel Project)
🔍 Overview

This project focuses on analyzing Netflix user data to uncover meaningful insights related to viewer behavior, content preferences, subscription trends, and customer satisfaction.

The dashboard is built using Microsoft Excel, leveraging data analysis and visualization techniques to transform raw data into actionable insights.

🎯 Objectives

Analyze user engagement patterns

Identify popular content genres

Understand subscription plan distribution

Track yearly subscription trends

Evaluate customer satisfaction levels

📁 Dataset Description

The dataset includes the following key attributes:

Customer ID

Subscription Date (Year, Month)

Region & City Tier

Age & Gender

Subscription Plan

Monthly Revenue (ARPU)

Preferred Genre

Watch Time

Device Type

Payment Method

Tenure

Churn Status

Satisfaction Score

🛠️ Tools & Techniques Used

Microsoft Excel

Data Cleaning

Pivot Tables

Data Aggregation

Dashboard Design

Slicers for interactivity

📊 Dashboard Features

Interactive Slicers for filtering:

Age Group

Subscription Plan

Viewer Type

Region

Visualizations included:

Viewer Engagement Distribution

Genre Preferences

Subscription Plan Analysis

Yearly Subscription Trends

Customer Satisfaction Scores

📌 Key Insights
👥 Viewer Behavior

Regular viewers form the largest segment

Very few casual users → indicates high engagement

🎬 Content Preferences

Drama and Documentary are the most preferred genres

👨‍💼 Demographics

Majority users belong to 25–45 age group (working professionals)

Senior viewers contribute a very small share

💳 Subscription Plans

Standard plan has the highest adoption

Premium plan adoption is lower (price-sensitive users)

📈 Trends

Peak subscriptions recorded in 2023

Slight decline observed afterward

⭐ Customer Satisfaction

Most users have ratings between 4–5

Indicates overall positive user experience

💡 Business Recommendations

Increase Premium plan adoption through:

Bundled offers

Exclusive content

Discounts for long-term users

Focus on top-performing genres (Drama & Documentary)

Maintain high satisfaction levels to reduce churn
